# eportfolio

This repository contains a **Hugo** (Wowchemy/Academic) e-portfolio for the **AI MSc** program at **University of Essex Online**.

## Repository structure

- [`/site`](./site): Website source (Hugo + Wowchemy/Academic)
- [`/src`](./src): Project code and supporting materials

## Deployment

The site is deployed to **GitHub Pages** via **GitHub Actions**.
